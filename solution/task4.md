# Задание 4
Виртуальные серверы — удобный инструмент для хранения данных и запуска веб-приложений. Арендуя такой сервер, пользователь получает изолированную виртуальную среду с собственной операционной системой и выделенными ресурсами. Однако физически эта машина не является отдельной: она делит вычислительные мощности (процессор, память, накопители) одного мощного сервера с другими виртуальными машинами.
