# Задание 6

 **Ключи доступа** — это набор учетных данных для аутентификации в сервисах Selectel. Они позволяют аутентифицировать пользователя или сервисные приложения без передачи основного пароля от аккаунта.
 В зависимости от задачи используются разные типы ключей:
 * **[SSH-ключи](https://docs.selectel.ru/account/manage-keys/#manage-ssh-keys)** — для подключения к облачным серверам по протоколу SSH. 
 * **[API-ключи](https://docs.selectel.ru/account/manage-keys/#manage-api-keys)** — для аутентификации при работе с API Selectel и автоматизации управления ресурсами.
 * **[S3-ключи](https://docs.selectel.ru/account/manage-keys/#manage-s3-keys)** — для аутентификации запросов к объектному хранилищу.
 * **[ADB-ключи](https://docs.selectel.ru/account/manage-keys/#manage-adb-keys)** — для подключения к устройствам мобильной фермы по ADB.
 
Ключи хранятся в вашем профиле и доступны во всех проектах, в которые вы добавлены. Управлять ими можно в панели управления в разделе **Аккаунт → Доступ**.
