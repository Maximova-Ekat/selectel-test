# Задание 2
**1. Уточню ожидания заказчика**

* кто **целевая аудитория статьи** — широкая аудитория, разработчики, инженеры, исследователи...
* какие **технологии** важно осветить (Android, компьютерное зрение, сбор телеметрии, архитектура системы);
* какие **конкретные примеры** можно использовать (кейсы из разработки, реальные задачи);

> Если материал рассчитан исключительно на широкую аудиторию, а заказчик хочет добавить глубокую техническую детализацию:
> объясню, что в данном случае ожидания не соответствуют принятому стайлгайду. Предложу компромиссный вариант: сохранить ключевые технические детали, которые важны для понимания системы, но подать их в более доступной форме. Например:
> * описать архитектуру и используемые технологии на уровне концепции;
> * добавить конкретные примеры работы системы, но описать их максильно доступным языком;
> * вынести более сложные технические детали в отдельные блоки, схемы или раздел «Подробнее».

 Если ожидания заказчика не противоречат друг другу, перейду к шагу 2.


**2. Переструктурирую текст**
Уберу лирические и описательные фрагменты и выстрою текст вокруг технической логики. Например, вокруг архитектуры программно-аппаратного комплекса и сравнения с архитектурой предшественников.


**3. Добавлю конкретные примеры**
Например:

* как Android-устройства используются для сбора данных с камер;
* как данные передаются в систему хранения;
* какие алгоритмы применяются для обработки изображений;
* какие данные собираются (координаты, изображения, временные метки).
Выделю кейсы, предложенные заказчиком.


**4. Усилю технологическую часть**
Добавлю:
* Названия протоколов и стандартов (LoRaWAN, GSM, GPS, ГЛОНАСС)
* Схемы взаимодействия компонентов;
* Конкретные технологии (FFT для анализа спектра, ResNet для классификации)
* Числовые параметры (частоты, веса, дальность, точность)
* Сравнительные таблицыи и диаграммы


**5. Согласую обновлённую версию**
Сначала согласую новую структуру и акценты, а затем доработаю текст до финальной версии.
